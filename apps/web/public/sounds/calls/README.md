# Call Sounds

These call tones are CC0/public-domain assets from BigSoundBank:

- `ringback.mp3`: Tone, Ringback Tone #3, sound 1617
- `ringtone.mp3`: Telephone booth ringtone #1, sound 3364
- `busy.mp3`: Tone, busy #1, sound 1610
- `ended.mp3`: Dial Tone, Call Ends, sound 2645

The app loads these files from `/sounds/calls/`.
